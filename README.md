Repository for work I'm doing to support [The Al'Kabor Project](https://www.takproject.net/).

Wayland-based Cast Script (bash script)
--------------------
This is an attempt to create a casting skill-up script based off Meriadoc's Xorg/X11 script: https://github.com/kai4785/takp/blob/master/skillup.sh

There are notes included in the script to help the technically less proficient. I'm going to try and create a launcher that will just accept input from the user, which will then call this script.

Lutris Install Script (YAML file)
--------------------
I've submitted the script to be included in the Lutris search for EverQuest; however, it seems like it can take awhile to get approved. Even then, I wasn't able to submit specifics, such as the `slug` name.

I will update this once/if it is approved.
